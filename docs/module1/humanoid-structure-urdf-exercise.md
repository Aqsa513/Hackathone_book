# Humanoid Structure with URDF - Hands-on Exercise

## Exercise: Creating a Simple Humanoid Robot Model

In this exercise, you'll create a basic humanoid robot model using URDF (Unified Robot Description Format).

### Prerequisites

- ROS 2 installed (Humble Hawksbill or newer)
- Basic understanding of XML syntax
- Understanding of robot kinematics concepts

### Task 2: Validating the URDF Model

1. Save the URDF code to a file named `simple_humanoid.urdf`

2. To validate the URDF file, you can use the check_urdf tool:
```bash
# First install the tool if not already available
sudo apt install ros-humble-urdf-tutorial

# Check the URDF file
check_urdf simple_humanoid.urdf
```

### Task 3: Visualizing the URDF Model

1. Create a launch file to visualize the robot in RViz:
```python
# launch/robot_visualization.launch.py
from launch import LaunchDescription
from launch.actions import DeclareLaunchArgument
from launch.substitutions import LaunchConfiguration
from launch_ros.actions import Node
from launch.substitutions import Command
from ament_index_python.packages import get_package_share_directory
import os

def generate_launch_description():
    use_sim_time = LaunchConfiguration('use_sim_time', default='false')

    urdf_file = os.path.join(
        get_package_share_directory('your_package_name'),
        'urdf',
        'simple_humanoid.urdf'
    )

    robot_description = {'robot_description': Command(['xacro ', urdf_file])}

    return LaunchDescription([
        DeclareLaunchArgument(
            'use_sim_time',
            default_value='false',
            description='Use simulation (Gazebo) clock if true'
        ),

        Node(
            package='robot_state_publisher',
            executable='robot_state_publisher',
            name='robot_state_publisher',
            output='screen',
            parameters=[robot_description, {'use_sim_time': use_sim_time}]
        ),

        Node(
            package='joint_state_publisher_gui',
            executable='joint_state_publisher_gui',
            name='joint_state_publisher_gui',
            output='screen'
        ),

        Node(
            package='rviz2',
            executable='rviz2',
            name='rviz2',
            output='screen'
        )
    ])
```

### Task 4: Running the Visualization

1. Create a URDF directory in your package:
```bash
mkdir urdf
```

2. Save your URDF file as `urdf/simple_humanoid.urdf`

3. Run the visualization:
```bash
ros2 launch your_package_name robot_visualization.launch.py
```

### Summary

This exercise demonstrates how to create a humanoid robot model using URDF. You've learned:
- How to define links with visual, collision, and inertial properties
- How to connect links using joints with different types and limits
- How to structure a complex robot model with multiple limbs
- How to validate and visualize your URDF model
