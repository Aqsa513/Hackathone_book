# Quiz: Cognitive Planning with LLMs

## Question 1
What is the primary role of LLMs in VLA systems?

A) To replace vision systems
B) To serve as the cognitive planning layer converting language to actions
C) To control robot motors directly
D) To process visual information

## Question 2
Which of the following is a best practice for LLM-based action planning?

A) Skip validation of generated action plans
B) Design clear prompt structures for consistent output
C) Ignore ambiguous commands completely
D) Execute plans without safety checks

## Question 3
What should happen when an LLM receives an ambiguous command?

A) Execute a random action
B) Ignore the command completely
C) Ask for clarification or provide reasonable interpretation
D) Stop the entire system permanently

## Question 4
How do LLMs convert natural language to ROS 2 actions?

A) Direct hardware control
B) Through cognitive planning and action sequence generation
C) By replacing ROS 2 entirely
D) Using only voice recognition

## Question 5
What is the relationship between language understanding and action execution?

A) No relationship exists
B) Language understanding drives action execution through planning
C) Actions control language understanding
D) They operate independently

## Answer Key
1. B - LLMs serve as the cognitive planning layer
2. B - Clear prompt structures ensure consistent output
3. C - Ambiguous commands should be handled with clarification
4. B - Through cognitive planning and sequence generation
5. B - Language understanding drives action execution through planning
